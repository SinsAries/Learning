Cho Q truy vấn, mỗi truy vấn có 2 tham số l và r, 
trả lời tổng các phần tử của mảng A từ vị trí l tới vị trí r

Code tham lam:
int tong = 0;
for (int i = l; i <= r; i++) {
    tong += A[i];
}
cout << tong;
