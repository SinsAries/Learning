Độ phức tạp thời gian

Giới thiệu

  • 1 giây có thể chạy khoảng 10810^8108 phép tính, chính xác hơn thì là gần 5∗1085 * 10^85∗108 phép tính

Tính toán độ phức tạp thời gian

  • Thường thì đầu vào sẽ có độ lớn là n
  • Độ phức tạp thời gian sẽ là số phép tính để chạy thuật toán trong tính huống xấu nhất
  • Độ phức tạp thời gian thường được kí hiệu bằng O(f(n))O(f(n))O(f(n))
  • Những code sau sẽ có độ phức tạp là O(1)O(1)O(1), hay chính xác hơn thường gọi là độ phức tạp hằng số
    int a = 10;
    int b = 7;
    int c = a + b + 10;
    cin >> a;
    cout << a;
  • Độ phức tạp của vòng lặp thì là số lần lặp lại mà vòng lặp lặp, ví dụ, độ phức tạp của những đoạn code sau là O(n)O(n)O(n) (quy ước những đoạn code bên trong vòng lặp sẽ có độ phức tạp là O(n)O(n)O(n))
    for (int i = 1; i <= n; i++) {
        // Code có độ phức tạp O(1)
    }

    int i = 0;
    while(i < n) {
        // Code có độ phức tạp O(1)
        i++;
    }

    for (int i = 1; i <= 5 * n + 17; i++) {
        // Code có độ phức tạp O(1)
    }

    for (int i = 1; i <= n + 231232; i++) {

    }
  • Chúng ta có thể tính độ phức tạp của những vòng lặp lồng vào nhau bằng cách nhân độ phức tạp của từng vòng lặp với nhau. Ví dụ sau đây có độ phức tạp là O(nm)O(nm)O(nm), vì vòng lặp bên ngoài chạy n lần và vòng lặp bên trong chạy mmm lần
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            // Code có độ phức tạp O(1)
        }
    }
  • Độ phức tạp trong ví dụ sau là O(n2)O(n^2)O(n2)
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            // Code có độ phức tạp O(1)
        }
    }
  • Độ phức tạp trong ví dụ sau là O(n2+m)O(n^2 + m)O(n2+m)
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            // Code có độ phức tạp O(1)
        }
    }

    for (int i = 1; i <= m; i++) {
        // Code có độ phức tạp O(1)
    }
